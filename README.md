# 음성 인식 기반 치매 위험도 분석 시스템

## 1. 프로젝트 개요

본 프로젝트는 사용자의 **일상 대화 음성**을 입력받아, 이를 텍스트로 변환한 뒤 언어적 특징을 분석하여 **치매 위험도를 평가하는 AI 기반 시스템**을 구현하는 것을 목표로 한다.  

기존의 치매 진단은 병원 방문, 설문, 전문 검사 등을 중심으로 이루어지는 경우가 많아 일상 속에서 조기 징후를 발견하기 어렵다. 이에 본 시스템은 **비침습적이고 접근성이 높은 대화 기반 분석 방식**을 활용하여, 사용자의 언어 패턴 속에서 나타날 수 있는 치매 전조 증상을 탐지하고자 한다.

본 시스템은 음성 인식(STT), 대화형 AI, 언어 패턴 분석, 위험도 시각화 기능을 통합하여 **실시간 인지 위험도 모니터링 시스템**의 형태로 구현되었다.

---

## 2. 프로젝트 필요성

고령화 사회로 진입하면서 치매 환자 수는 지속적으로 증가하고 있으며, 치매의 조기 발견과 초기 대응의 중요성 또한 커지고 있다. 치매는 초기 단계에서 일상 대화 속에 다음과 같은 언어적 특징으로 나타날 수 있다.

- 같은 질문을 반복하는 현상
- 기억이 나지 않는다는 표현 증가
- 시간이나 상황에 대한 혼란
- 문장의 응집성 저하 및 비논리적 표현

이러한 특징은 단순 대화 속에서도 관찰될 수 있으므로, 음성과 대화 데이터를 기반으로 분석할 수 있는 AI 시스템은 **조기 선별 도구**로서 의미가 있다.  
따라서 본 프로젝트는 사용자의 발화를 분석하여 위험도를 수치화하고, 이를 직관적으로 시각화함으로써 실질적인 보조 시스템을 제공하고자 한다.

---

## 3. 프로젝트 목표

본 프로젝트의 목표는 다음과 같다.

1. 사용자의 음성을 텍스트로 변환할 수 있는 음성 인식 기능 구현
2. 대화형 AI를 통해 사용자와 자연스러운 상호작용 제공
3. 대화 내용 속 언어적 특징을 분석하여 치매 위험도 점수 산출
4. 분석 결과를 그래프 및 카드 형태로 시각화
5. 기억력 확인을 위한 Recall Memory Test 기능 제공

즉, 단순한 챗봇 구현이 아니라 **대화 데이터를 기반으로 인지 상태를 분석하는 통합형 AI 시스템**을 구현하는 것이 핵심 목표이다.

---

## 4. 시스템 구성

본 시스템은 크게 다음과 같은 흐름으로 동작한다.

```text
사용자 음성 입력
        ↓
Speech-to-Text 변환
        ↓
텍스트 기반 AI 응답 생성
        ↓
언어 패턴 분석
        ↓
치매 위험도 점수 산출
        ↓
시각화 및 기록 저장
```
### 세부 구성
- **음성 입력부**: 사용자의 음성을 녹음
- **STT 모듈**: 음성을 텍스트로 변환
- **대화 응답 모듈**: 사용자 질문에 간단하고 직접적인 응답 생성
- **위험도 분석 모듈**: 언어 패턴 기반 점수 산출
- **Recall Memory Test 모듈**: 단기 기억력 확인
- **시각화 모듈**: 위험도, 추세, 특징 점수 등을 그래프로 표현

## 5. 주요 기능

### 5.1 음성 인식 기능
사용자의 음성을 입력받아 텍스트로 변환한다.  
이를 통해 키보드 입력 없이도 시스템 사용이 가능하며, 실제 고령층 사용 환경을 고려한 인터페이스를 제공한다.

### 5.2 AI 대화 응답 기능
입력된 텍스트에 대해 LLM 기반의 응답을 생성한다.  
응답은 짧고 명확한 정보 전달을 목표로 하며, 사용자와의 자연스러운 상호작용을 지원한다.

### 5.3 치매 위험도 분석 기능
대화 내용을 바탕으로 다음 4가지 언어 특징을 분석한다.

- **질문 반복**
- **기억 혼란 표현**
- **시간/상황 인식 혼란**
- **문장 비논리성**

각 항목별 점수를 합산하여 최종 위험도 점수를 0~100점 범위로 산출한다.

### 5.4 위험도 시각화 기능
분석 결과를 사용자가 직관적으로 이해할 수 있도록 시각화한다.

- 전체 평균 점수
- 최근 평균 점수
- 최근 점수
- 시간별 점수 추이 그래프
- 위험도 게이지 차트
- 언어 특징 Radar Chart

### 5.5 Recall Memory Test
시스템이 특정 단어를 제시한 뒤, 이후 대화에서 해당 단어를 기억하는지 확인하는 기능이다.  
이를 통해 단순 언어 패턴 분석 외에 **단기 기억력 확인 요소**를 추가하였다.

### 5.6 위험 경고 기능
위험 점수가 일정 기준 이상일 경우 경고 팝업을 표시하여 사용자가 현재 상태를 인지할 수 있도록 한다.

## 6. 분석 기준

본 시스템은 대화 속 언어적 특징을 기반으로 점수를 산출한다.

| 분석 항목 | 설명 | 점수 범위 |
|----------|------|-----------|
| 질문 반복 | 같은 질문 또는 유사 의미 질문 반복 여부 | 0 ~ 25 |
| 기억 혼란 | "기억이 안 난다", "뭐였지" 등의 표현 | 0 ~ 25 |
| 시간/상황 혼란 | 날짜, 시간, 현재 상황 인식 오류 | 0 ~ 30 |
| 문장 비논리성 | 문장 구조 불안정, 응집성 저하 | 0 ~ 20 |

### 종합 점수 기준

| 점수 구간 | 위험도 판단 |
|----------|-------------|
| 0 ~ 19 | 정상 |
| 20 ~ 39 | 약한 의심 |
| 40 ~ 59 | 중간 의심 |
| 60 ~ 79 | 높은 의심 |
| 80 ~ 100 | 매우 높은 의심 |

## 8. 기대 효과

본 프로젝트를 통해 다음과 같은 효과를 기대할 수 있다.

1. **일상 대화 기반 조기 선별 가능성 제시**  
   병원 검사 이전 단계에서 위험 징후를 탐지할 수 있다.

2. **비침습적 건강 모니터링 시스템 구현**  
   사용자는 자연스러운 대화만으로 분석 결과를 얻을 수 있다.

3. **직관적인 데이터 시각화 제공**  
   분석 결과를 점수와 그래프로 확인할 수 있어 이해가 쉽다.

4. **확장 가능한 헬스케어 시스템 기반 마련**  
   향후 감정 분석, 영상 분석, 생체 데이터와 결합 가능한 구조를 가진다.

## 9. 결론

본 프로젝트는 음성 인식과 자연어 처리 기술을 활용하여, 사용자의 일상 대화 속 언어 패턴을 분석하고 치매 위험도를 수치화하는 시스템을 구현하였다.  

특히 단순 응답형 챗봇이 아니라,  
**대화 → 분석 → 점수화 → 시각화**의 구조를 갖춘 인지 위험도 모니터링 시스템이라는 점에서 의의가 있다.  

향후 데이터 확장과 모델 고도화를 통해 실제 헬스케어 분야에서 활용 가능한 보조 시스템으로 발전할 가능성이 있다.
